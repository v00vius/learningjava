Напишите программу, которая должна имитировать раздачу карт для игры в покер.
Программа получает число n, задаваемое с консоли пользователем,
и раздает карты на n игроков (по 5 карт каждому) из рассортированной колоды.
Разделяйте пять карт, выданных каждому игроку, пустой строкой.


====================================================

Описание нашего приложения:

1. Класс Card - который нам нужен для как элемент нашей карточной колоды, из которой мы будем раздавать карты игрокам.
   Будет содержать два параметра: номинал карты и масть

2. Класс Cards - в котором будет храниться наша колода, т.е. 52 экземпляра типа Card

3. Класс Player - будет создаваться для каждого игрока
    Его параметры:
    - имя игрока
    - массив из 5 элементов типа Card в котором будут хранится ссылки на те экземпляры классов Card, которые мы "раздали" этому игроку


Запуск приложения.
- подготовительные действия с колодой карт
    - создать колоду (создать массив)
    - наполнить (создать необходимое количество экземпляров класса Card)
    - перемешать

    - ввод количества игроков

    - ввести данные игроков

    - раздать карты игрокам

    - вывести информацию о картах у игроков



    колода карт - массив из 52 карт (4 масти по 13 карт от 2 до туза)
